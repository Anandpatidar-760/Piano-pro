This is a web application that enables playing the piano when a real one is not handy.
This is built using HTML, CSS,Bootstrap and JavaScript.
This application is Devloped and Desifned By Anand Patidar.

Features:-
Piano with all 24 keys having diffrent sounds.
3D pressing effects on key click.
Feel like real piano.
Produce sounds like basic piano.
Unique and dynaic project.
Unique CSS effects.
To practise JS DOM.
